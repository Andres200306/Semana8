Qué muestra cada comando:
- `uname -a`: muestra información del kernel, nombre del host, versión del kernel, fecha de compilación y la arquitectura del sistema.
- `whoami`: muestra el nombre del usuario con el que se está ejecutando la sesión actual.

Por qué es útil conocer esta información al administrar un servidor:
- Permite identificar la distribución y versión del kernel para aplicar actualizaciones o parches correctos.
- Ayuda a diagnosticar problemas de compatibilidad de software y arquitectura (por ejemplo, x86_64 vs ARM).
- Conocer el usuario actual es importante para verificar permisos y confirmar en qué cuenta se realizan cambios administrativos.
